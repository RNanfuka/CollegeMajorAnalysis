# Feedback and Contribution
We welcome any input or feedback on [Adult Income Prediction](https://github.com/RNanfuka/AdultIncomePrediction/new/main). In particular, we welcome people to propose ideas and submit pull request to our analysis.

Here are the steps to contribute. The instructions below are provided by Copilot by "Can you please provide me step-by-step instructions how a stranger can get submit pull requests to my public GitHub repo?"

1. Ask the GitHub repo owner to add you as collaborators.
2. Clone the repo to your local:
```cmd
git clone git@github.com:RNanfuka/AdultIncomePrediction.git
```

3. Check out a new feature name, make changes, and commit.
```cmd
git checkout -b feature/feature-name
```

4.  Push your own branch up.
```cmd
git push origin feature/short-descriptor
```

6. Open a pull request on GitHub and add one of us as the reviewer.
